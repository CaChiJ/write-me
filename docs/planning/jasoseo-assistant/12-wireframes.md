# 12. Wireframes

## 목적
이 문서는 구현 가능한 수준의 high-fidelity ASCII wireframe을 제공한다.  
실제 UI 라벨, 패널 구조, 상태 차이, 행동 버튼 위치를 고정한다.

## 공통 레이아웃 키
- `NAV`: 좌측 글로벌 내비게이션 rail
- `SRC`: 자료 패널
- `EDT`: 본문 에디터
- `AI`: 도구 메뉴 + AI 채팅 패널
- `BAR`: sticky action bar

## 와이어프레임 검토 체크
- 모든 desktop 화면은 좌측 rail 기준 정렬을 유지한다.
- 에디터의 primary action은 항상 화면 하단 sticky bar에 남긴다.
- mobile에서는 좌/우 패널이 직접 보이지 않고 sheet 또는 탭 액션으로 이동한다.
- unresolved `INFERRED`는 편집 화면과 리뷰 화면 모두에서 명시적으로 노출된다.
