# 신입 개발자 자기소개서 보조 서비스 기획서 (Index)

## 문서 목적
- 신입 개발자의 다양한 자기소개서 작성 상황을 **하나의 직관적인 UX**로 포괄하는 제품 기획 초안
- 지원 페이지 텍스트를 붙여넣으면 형식/문항/글자수를 자동 구조화하는 통합 작성 경험 제공
- 사용자 초안 편집 방식과 보유 자료 기반 생성 방식을 동시에 지원
- 본 문서는 기술 구현 상세 전 단계의 **제품/UX 중심 기획** 문서
